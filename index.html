HTML Viva Voce (40 Questions)

1. What does <!DOCTYPE html> do?

Tells the browser to render in Standard Mode (HTML5) instead of Quirks Mode.

2. Semantic vs Non-Semantic?

Semantic elements (<header>) describe meaning. Non-semantic (<div>) do not.

3. Block vs Inline?

Block starts on new line (full width). Inline stays on same line (content width).

4. Void Elements?

Tags with no closing tag and no content (e.g., <img>, <br>, <hr>).

5. ID vs Class?

ID is unique per page. Class can be reused on multiple elements.

6. What is the <head> tag?

Container for metadata (title, styles, scripts) not shown in viewport.

7. What is <meta charset="UTF-8">?

Sets character encoding to UTF-8 to support all human languages/symbols.

8. Global Attributes?

Attributes valid on any tag: id, class, style, title, hidden, tabindex.

9. GET vs POST?

GET retrieves data (in URL). POST submits data (in body, secure).

10. <label> tag purpose?

Improves accessibility; clicking label focuses the linked input.

11. HTML5 Data Attributes?

Store custom data (data-*) accessible via JS dataset API.

12. What is <fieldset>?

Groups related elements in a form, used with <legend>.

13. src vs href?

src embeds resource (img/script). href links to resource (anchor/css).

14. alt attribute importance?

Critical for Screen Readers (accessibility) and SEO.

15. Canvas vs SVG?

Canvas is raster (pixels, JS drawn). SVG is vector (XML, scalable).

16. What is <picture>?

Allows art direction (different images for different screens).

17. async vs defer?

Async runs immediately (pauses HTML). Defer runs after HTML parsing.

18. Local vs Session Storage?

Local persists forever. Session clears when tab closes.

19. What is <iframe>?

Embeds another HTML page within the current page.

20. Viewport Meta Tag?

Controls layout on mobile (width=device-width).

21. <strong> vs <b>?

Strong is semantic importance. B is just visual bold.

22. <em> vs <i>?

Em is semantic emphasis. I is just visual italic.

23. What is <noscript>?

Content displayed only if JS is disabled.

24. download attribute?

Prompts browser to download the link instead of navigating.

25. <details> tag?

Native accordion widget with expand/collapse capability.

26. <base> tag?

Sets default URL for all relative links in document.

27. contenteditable?

Attribute that makes any element editable by user.

28. autocomplete attribute?

On/Off. Allows browser to suggest values for inputs.

29. <datalist> tag?

Provides autocomplete suggestions for input elements.

30. <map> tag?

Defines an image map with clickable areas.

31. <mark> tag?

Highlights text (usually yellow background).

32. <time> tag?

Represents a specific period in time (machine readable).

33. What is Favicon?

Small icon displayed in browser tab (linked in head).

34. <abbr> tag?

Defines abbreviation; title attr shows full text on hover.

35. <blockquote> vs <q>?

Blockquote for block-level quotes. Q for inline quotes.

36. <sub> and <sup>?

Subscript (H2O) and Superscript (X2).

37. <pre> tag?

Preserves both spaces and line breaks in text.

38. <code> tag?

Defines computer code text (monospaced font).

39. hidden attribute?

Global attribute that hides element (like display:none).

40. autofocus attribute?

Input automatically gets focus when page loads.

CSS Viva Voce (40 Questions)

1. What is CSS Box Model?

Every element is a box: Content, Padding, Border, Margin.

2. box-sizing: border-box?

Includes padding/border in total width/height calculations.

3. CSS Specificity?

Inline > ID > Class > Element. !important overrides all.

4. display:none vs visibility:hidden?

None removes from flow. Hidden keeps space but invisible.

5. What is z-index?

Controls stack order of positioned elements.

6. Reset vs Normalize?

Reset strips all styles. Normalize makes defaults consistent.

7. Pseudo-classes vs Elements?

Classes (:hover) define state. Elements (::before) style parts.

8. px vs em vs rem?

px=absolute. em=relative to parent. rem=relative to root html.

9. position: absolute?

Positioned relative to nearest positioned ancestor.

10. position: fixed?

Positioned relative to the viewport (stays on scroll).

11. position: sticky?

Toggles between relative and fixed based on scroll pos.

12. What is Flexbox?

1D layout model for rows OR columns. Good for components.

13. What is CSS Grid?

2D layout model for rows AND columns. Good for page layout.

14. Center div (Flex)?

display:flex; justify-content:center; align-items:center;

15. justify-content vs align-items?

Justify=Main Axis (Row). Align=Cross Axis (Column).

16. What is `gap`?

Space between flex/grid items (replaces margin hacks).

17. Mobile First Design?

Style for mobile first, then use min-width media queries.

18. Media Queries?

Apply styles only if condition met (@media (max-width:700px)).

19. CSS Variables?

--color: red; usage: var(--color). Reusable values.

20. opacity vs rgba alpha?

Opacity fades element+children. RGBA fades only background.

21. object-fit: cover?

Image fills container, cropping excess to maintain aspect ratio.

22. vh and vw units?

1vh = 1% of viewport height. 1vw = 1% of viewport width.

23. Transition vs Animation?

Transition=A to B on trigger. Animation=Keyframes, loops, complex.

24. transform property?

Rotate, Scale, Skew, Translate without affecting flow.

25. clip-path?

Clips an element to a basic shape (circle, polygon).

26. ::placeholder?

Pseudo-element to style input placeholder text.

27. pointer-events: none?

Element ignores clicks; clicks pass through to behind.

28. will-change?

Performance hint to browser about upcoming changes.

29. user-select: none?

Prevents user from selecting/highlighting text.

30. backdrop-filter?

Blur/effect behind the element (Glassmorphism).

31. calc() function?

Performs calcs mixing units (width: calc(100% - 20px)).

32. @font-face?

Loads custom fonts into the webpage.

33. outline vs border?

Outline doesn't take space and sits outside border.

34. currentColor?

Variable holding the element's current text color.

35. flex-grow?

How much a flex item grows relative to others.

36. flex-shrink?

How much a flex item shrinks if space is limited.

37. :nth-child()?

Selects elements based on their index (odd, even, 3n).

38. :not() selector?

Selects elements that do NOT match criteria.

39. > vs space selector?

> is direct child. Space is any descendant.

40. float and clearfix?

Float pushes element side. Clearfix fixes parent collapse.
